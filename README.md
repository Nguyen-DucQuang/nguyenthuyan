# 🌸 Portfolio Website – Nguyễn Thúy An

> Một portfolio cá nhân hiện đại, tinh tế và mang phong cách nhẹ nhàng dành cho cô **Nguyễn Thúy An**.

---

## 📌 Giới thiệu

Đây là dự án Portfolio cá nhân được xây dựng nhằm giới thiệu thông tin, kỹ năng, dự án và phong cách cá nhân của cô **Nguyễn Thúy An** thông qua giao diện hiện đại, tối giản và thân thiện với người dùng.

Website được tập trung vào:

* ✨ Thiết kế thanh lịch, mềm mại
* 📱 Responsive trên mọi thiết bị
* ⚡ Hiệu ứng mượt mà
* 🎨 Tối ưu trải nghiệm người dùng
* 💻 Code sạch và dễ mở rộng
